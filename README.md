# System Monitor in Linux using C++

A simple system-monitoring tool in C++ that displays:
- CPU usage
- Memory usage
- Running processes
- Memory per process
- Kill a process using PID
- Sorting by memory

### Features
- Display overall system stats
- Manual refresh
- List running processes
- Sort by memory
- Kill process
- Minimal, terminal-based UI

### Requirements
- Linux / WSL
- g++
- C++17+
